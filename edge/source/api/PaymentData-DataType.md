###PaymentData.DataType

The DataType enum identifies the asynchronous method for which the response is received.

**Values:**

* **ENABLE**

* **READCARDDATA**

* **PROMPTPIN**

* **PROMPTMENU**

* **PROMPTMESSAGE**

* **GETEMVTAG**

* **PROMPTADDITIONALINFO**

* **COMPLETEONLINEEMV**

* **VALIDATEMAC**

* **AUTHORIZECARD**

* **UNDEFINED**
