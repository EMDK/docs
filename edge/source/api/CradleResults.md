###CradleResults

Defines all the error codes the cradle API returns.

**Values:**

* **SUCCESS**

* **FAILURE**

* **ALREADY_ENABLED**

* **ALREADY_UNLOCKED**

* **ALREADY_DISABLED**

* **ALREADY_IN_USE**

* **FEATURE_NOT_SUPPORTED**

* **INVALID_VALUE**

* **NOT_ENABLED**

* **INVALID_OBJECT**

* **UNDEFINED**
